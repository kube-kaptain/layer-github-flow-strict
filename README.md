# Layer GitHub Flow Strict

Kaptain config layer enforcing strict GitHub flow quality checks.

Projects that reference this layer inherit:

- **Branch name slash blocking** — no slashes in branch names, poor taste
- **Conventional commit blocking** — no conventional commit format allowed

See [GitHub Flow](https://docs.github.com/en/get-started/using-github/github-flow) for the branching model this layer enforces.
